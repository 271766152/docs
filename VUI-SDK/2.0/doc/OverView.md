
VUISDK_V2.0 简介
=

**VUI SDK是roobo基于Android平台推出的一款包含基础语音、语义能力的软件开发包，SDK连接了roobo的语音、语义服务平台以及内容平台。VUI SDK为客户提供了：**
  
-  语音端点检测,语音边界检测的能力。  
-  离线语音识别和在线语音识别的能力。  
-  离线TTS和在线TTS的能力。  
-  丰富的语义和内容服务。
