
VUISDK_V2.0 简介
=

**VUI SDK是roobo结合自身机器人和语音对话系统开发经验推出的一款包含基础语音能力和语音交互范式的软件开发包。VUI SDK连接roobo的语音语义服务平台，给客户提供完整的语音系统开发能力支持。 
VUI SDK为客户提供了：**
  
-  AEC前端信号处理。  
-  语音端点检测,语音边界检测的能力。  
-  离线语音识别和在线语音识别的能力。  
-  离线TTS和在线TTS的能力。  
-  丰富的语义和内容服务。
